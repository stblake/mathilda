# Graphs

2 built-in function(s) in this category.

- [`Graph`](Graph.md) — Graph[v, e] represents a graph with vertices v and edges e. Graph[e] derives the vertices from the edge list. Edges are DirectedEdge[u,v] or UndirectedEdge[u,v]; u->v and u<->v are accepted as shorthand. Simple graphs only: no self-loops or parallel edges.  _(Stable)_
- [`GraphQ`](GraphQ.md) — GraphQ[g] gives True if g is a valid graph, and False otherwise.  _(Stable)_
