## Algorithm

### Primitive A — Integer/Rational Log decomposition
For every `Log[n]`/`Log[p/q]` with positive rational arg, replace by
Σ ci Log[pi] using prime factorisation. Cheap trial division up to ~10^12.
Soundness: unconditional.

### Primitive B — Linear-combination-of-logs fuser
Given a Plus = Σ ci Log[ai] + R, fuse log block into single `Log[∏ ai^ci]`
when fused arg simplifies. Try `Together` on the product, evaluate, and if
result is `1` the entire log block vanishes. Soundness: requires positive
ai, but for symbolic algebraic args we allow it under positivity assumption.

### Pipeline placement
Insert into `simp_pipeline_logexp` between `apply_logexp_rules` and
`apply_assumption_rules` as a force-take seed.
